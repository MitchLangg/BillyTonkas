# BillyTonkas
Billy Tonka's Fudge Factory is a JavaFX-MySQL candy Inventory management software.

# Features
- Login/ New Account functionality
- Passwords are stored in a file i/o text file.
- Add Items
- Delete Items
- Update Items
- Display Items
- Pie Charts displaying the stock
